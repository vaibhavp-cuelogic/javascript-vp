Day 2 Assignments

1. Write a program to create functions that will do the following

1. Add 2 numbers
2. Subtract 2 numbers
3. Divide 2 numbers
4. Multiply 2 numbers

NOTE: For above use doOperation(operation,param1,param2) function as example. Where operation will be from below list:

1.add

2.subtract

3.multiply

4.divide

And param1, param2 will be any numbers such as 5,5

Example: doOperation('add',5,5)

Then update the program to get dynamic no of arguments(there can be 2,3,4 etc no of variables passed in when invoking the functions) The function should accept at least 2 variables and throw error if less then 2 are passed. The function should also validate the type of argument passed in

NOTE: For above use doOperationDynamic(operation,param1,param2,param3,param4,....) function as example. Where operation will be from below list:

1.add

2.subtract

3.multiply

4.divide

And param1, param2,... will be any numbers such as 5,10,...

Example: doOperationDynamic('subtract',100,25,25)
